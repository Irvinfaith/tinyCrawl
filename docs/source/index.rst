############################
tinyCrawl Documentation
############################

tinyCrawl 是一个微型轻量的爬虫框架，具有以下特点：

- 简单轻巧，没有任何第三方包的依赖
- checkpoint断点续爬
- 支持多线程爬取
- 内置日志功能
- 使用简单

`项目地址 <https://github.com/Irvinfaith/tinyCrawl>`_
